---

## Règles pour les descriptions Vinted

1. **Jamais de marque inventée** dans la description (sauf si étiquette physique)
2. **Toujours mentionner la taille** + équivalence numérique
3. **Toujours "envoi rapide"** → signal de confiance
4. **3-5 hashtags** pertinents → aide le référencement Vinted
5. **Ton naturel** — comme une vraie personne qui vend, pas un robot
6. **Varier les descriptions** d'un produit à l'autre (anti-détection)
7. **Pas de lien**, pas de mention d'autres plateformes
8. **Pas de prix dans la description** (c'est dans le champ prix)

---

## Variantes de formulation (pour éviter la répétition)

**Ouvertures :**
- "Très joli [produit] en [couleur]"
- "[Produit] [couleur], parfait pour [occasion]"
- "Je vends ce [produit] [adjectif]"
- "Magnifique [produit] style [niche]"

**États :**
- "Jamais porté, neuf sans étiquette"
- "Porté une seule fois, comme neuf"
- "En très bon état, aucun défaut"
- "État impeccable, pas de trace d'usure"

**Fermetures :**
- "Envoi rapide sous 24-48h"
- "Expédition rapide, emballage soigné"
- "Envoi le jour même si commandé avant 16h"
- "N'hésitez pas à me contacter pour plus d'infos"

## Comment utiliser avec Miggy

Envoie-moi :
1. Le type de produit (haut, pantalon, ensemble, accessoire)
2. La niche / le style (corporate, streetwear, old money, etc.)
3. Les couleurs
4. La taille
5. Le prix de vente prévu

→ Je te génère 1 description unique, prête à coller.

Pour du batch : envoie-moi une liste de 5-10 produits et je te génère toutes les descriptions d'un coup.
