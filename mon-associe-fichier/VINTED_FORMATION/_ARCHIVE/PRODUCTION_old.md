# PRODUCTION — Créer et publier une annonce (ARCHIVÉ)

> Ce fichier est archivé. Le process complet et à jour est dans PROCESS/PRODUCTION_IMAGES.md

---

## Workflow de publication

```
1. Choisir le produit (voir SOURCING.md)
2. Choisir le compte cible (selon niche)
3. Générer / préparer les photos
4. Rédiger la description (voir PROMPT_DESCRIPTIONS.md)
5. Publier depuis téléphone (pas PC)
6. Vérifier que l'article est actif (non masqué) sous 24h
7. Si article performant (+10 fav) → envisager boost
```

---

## Règles photos (anti-masquage)

- Photographier depuis le téléphone directement (pas de retouche PC)
- Fond naturel : parquet, lit, canapé — pas de fond blanc uni studio
- Plusieurs angles : face, dos, détail matière, ambiance
- Résolution standard téléphone (pas de rehaussement IA)
- Aucune retouche avec outil IA (pas de background remover IA)
- Si on utilise photos fournisseur : vérifier les EXIF, supprimer les métadonnées IA

---

## Règles description

- Longueur : 3-5 lignes
- Mentionner matière, couleur exacte, état (neuf), dimensions si pertinent
- Pas de majuscules abusives ou de spam de mots-clés

---

## Rythme de publication par compte

| Règle | Détail |
|-------|--------|
| Max publications/jour | 3 articles par compte |
| Espacement | Au moins 2h entre chaque article |
| Nouveau compte | Commencer par 2-3 articles sur 3 jours |
| Relance | Republier un article peu vu tous les 7-10 jours |
