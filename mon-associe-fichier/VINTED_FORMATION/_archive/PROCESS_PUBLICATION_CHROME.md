# PROCESS PUBLICATION — Claude in Chrome sur Vinted

> Mode opératoire pour poster des annonces Vinted via Claude

---

## Prérequis

1. Chrome Dev installé avec profils créés
2. Compte Vinted connecté sur le profil Chrome actif
3. Photos du produit prêtes (face + dos, ratio 4:3, format JPG/PNG)
4. Description générée par Miggy
5. Prix de vente décidé

---

## Workflow par annonce

### Étape 1 — Ouvrir Vinted
- Naviguer vers vinted.fr
- Vérifier que le bon compte est connecté

### Étape 2 — Cliquer "Vends tes articles"
- Bouton en haut de page

### Étape 3 — Upload photos
- Uploader photo face (photo 1 = cover)
- Uploader photo dos (photo 2)
- Uploader photo détail si disponible (photo 3)

### Étape 4 — Remplir les champs
| Champ | Valeur |
|---|---|
| Titre | [Nom produit] - [Couleur] - Taille [X] |
| Description | Description générée par Miggy |
| Catégorie | Sélectionner la bonne catégorie |
| Marque | "Autre" ou marque si applicable |
| Taille | Selon le produit |
| État | "Neuf sans étiquette" ou "Très bon état" |
| Couleur | Sélectionner |
| Prix | Prix de vente décidé |

### Étape 5 — Publier
- Cliquer "Ajouter"
- Vérifier que l'annonce est en ligne

### Étape 6 — Pause
- Attendre 3-5 minutes avant la prochaine annonce
- Rythme humain = anti-détection

---

## Rythme recommandé

| Phase | Annonces/session | Durée | Fréquence |
|---|---|---|---|
| Début (test) | 5-10 | 30-60 min | 1×/jour |
| Validation | 10-15 | 45-75 min | 1×/jour |
| Scale | 15-20 | 60-90 min | 1-2×/jour |

---

## Anti-détection

- **Varier les descriptions** (jamais copier-coller identique)
- **Varier les intervalles** (pas exactement 3 min à chaque fois, entre 2 et 5 min)
- **Photos uniques par annonce** (pas les mêmes photos sur 2 comptes)
- **1 niche par compte** (cohérence = naturel)
- **Ne pas poster 50 annonces d'un coup** (max 15-20 par session)

---

## Gestion multi-comptes (séquentiel)

1. Session sur Profil Chrome 1 → Compte Vinted 1 → 5-10 annonces
2. Pause 10 min
3. Switch sur Profil Chrome 2 → Compte Vinted 2 → 5-10 annonces
4. Etc.

⚠️ Claude in Chrome ne peut pas gérer plusieurs profils en parallèle.
Un profil à la fois, en séquence.

---

## Ce que Yanis prépare avant chaque session

1. **Dossier de photos** organisé par produit (ex: /produit_01/face.jpg, dos.jpg)
2. **Liste de produits** avec : nom, catégorie, taille, couleur, prix
3. **Descriptions** générées par Miggy en batch

→ Claude in Chrome n'a plus qu'à exécuter : ouvrir, uploader, remplir, publier, pause, suivant.
