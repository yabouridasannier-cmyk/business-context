## Le process complet en 5 étapes

### Étape 1 — Trouver le produit (vidéo niche)
Voir fichier `03_NICHES.md`

### Étape 2 — Trouver les photos source

**Principe** : Ne JAMAIS prendre les photos AliExpress directement → strike Vinted immédiat.

**Méthodes pour trouver des photos "portées" :**

1. **Avis des marketplace** — Regarder les photos clients dans les avis AliExpress/Temu/1688
2. **1688.com** — Recherche par image → versions chinoises avec photos portées, articles à ~5€ vs 20€ AliExpress
3. **Google Images** — Recherche image inversée à partir de la photo produit
4. **TikTok** — Screenshot de vidéos de mode (crops rapides)
5. **Pinterest** — Photos d'inspiration directement utilisables
6. **Boutiques dropshipping concurrentes** — Si tu trouves un site dropshipping qui vend le même produit, ils ont déjà fait la recherche produit pour toi → prendre leurs photos comme base

> "Si tu trouves des sites dropshipping qui vendent le même produit, tu as gagné. Ils font toute la recherche produit pour toi."

## Technique Gemini (recommandée pour débuter)

1. Trouver une **photo de référence** sur Pinterest (pose + fond intéressant)
2. Envoyer à Gemini : "Make the woman in first picture wear the [article] in photo 2"
3. Pouce en l'haut pour valider → Gemini enregistre comme référence
4. Envoyer le prompt d'itération : "Maintenant c'est notre photo de référence, ton travail va être de la faire porter tous les habits que je vais t'envoyer"
5. Pour les photos de dos : "Pour toutes les photos suivantes, mets la dame de dos en respectant toutes les caractéristiques morphologiques humaines"
6. Envoyer chaque nouveau produit → photo générée automatiquement

**Avantage clé** : Homogénéité — toujours le même fond pour toutes les annonces du même compte.

## Technique ChatGPT

1. Trouver un fond sur Pinterest
2. Demander à ChatGPT : "Réalise un prompt pour générer une photo destinée à Vinted, le plus naturel et réaliste possible, utilisant le fond de cette photo"
3. Obtenir le prompt détaillé avec positionnement + fond
4. Générer les photos → cropez pour bien cadrer le produit
5. Si trop lisse → demander d'ajouter du grain, réduire le lissage

## Règles de cadrage Vinted (CRITIQUE)

### Ratios obligatoires
- **Affichage annonce** : ratio 4:2.5 (rectangle vertical)
- **Affichage sur le profil/compte** : ratio 4:3
- **Toujours en rectangle**, jamais en carré

### Cadrage par type de produit

| Type | Cadrage | Zone |
|---|---|---|
| **Hauts** | Mi-cuisse → Tête | Personne centrée, selfie face |
| **Pantalons** | Hanche → Pieds (avec espace) | 3/4, pieds visibles (montre comment ça tombe) |
| **Accessoires/Objets** | Objet centré, fond neutre | Pas de déco, pas de couleurs flashy |

### Ce qui fait vendre : la règle RND
- **R**éaliste — pas de fond blanc pro, on veut "chez soi"
- **N**aturel — pas de mannequin figée, position selfie décontractée
- **D**émarqué — par la qualité, la couleur OU l'originalité du fond

### Ce qui fait FUIR (à éviter absolument)
- ❌ Photos sur fond blanc type e-commerce
- ❌ Mannequin dans une pose non naturelle
- ❌ Photo mal cadrée (trop de vide en haut)
- ❌ Photo trop sombre, sans lumière
- ❌ Tête visible entièrement (risque + pas naturel)
- ❌ Article non centré

### Ce qui MARCHE
- ✅ Selfie miroir dans une chambre/salon
- ✅ Fond coloré original (mur rose, mur marron)
- ✅ Cadrage serré sur le produit
- ✅ Main visible tenant le téléphone → authenticité
- ✅ Fond homogène sur tout le compte
