> Envoie-moi : prix achat + prix vente → je te sors tout

## Formule complète

```
MARGE NETTE = Prix vente - Commission Vinted (5%) - Prix achat - [Warehouse 2,20€] - URSSAF (~22%)
```

## Grille rapide (AVEC warehouse M22)

Ajouter 2,20€ par vente + 50€/mois fixe (réparti sur le nombre de ventes)

| Ventes/mois | Coût warehouse/vente | Impact sur marge |
|---|---|---|
| 50 | 2,20 + 1,00 = **3,20€** | Fort |
| 100 | 2,20 + 0,50 = **2,70€** | Modéré |
| 200 | 2,20 + 0,25 = **2,45€** | Faible |
| 300 | 2,20 + 0,17 = **2,37€** | Faible |

---

## Seuils de rentabilité

- **x3 minimum** → survie (marge faible après impôts)
- **x5** → confortable, objectif de base
- **x10+** → excellent, permet le scaling et les erreurs
- En dessous de x3 → ne pas vendre ce produit
