## Les 3 façons de trouver une niche

1. **Expertise personnelle** — tu aimes les sacs, tu vends des sacs. Tu connais déjà le marché.
2. **Recherche internet** — tu cherches ce qui marche et tu le vends.
3. **Techniques de veille** (détaillées ci-dessous)

---

## Techniques de veille pour trouver une niche

### Technique 1 — Pinterest
- Créer un compte Pinterest en te déclarant **femme** + intérêts : look, maquillage, culture pop, esthétique, tenues tendance
- Se balader dans le feed → noter ce qui revient souvent
- Cliquer sur un style → le feed s'enrichit automatiquement sur ce style
- Si tu vois des gens qui se "cassent le cul" à faire des photos → il y a de la demande

### Technique 2 — TikTok
- Taper des requêtes comme "outfit woman", "fit automne", etc.
- Regarder les likes → 452 likes = 452 personnes qui veulent ce produit
- Si les gens demandent "c'est quoi la ref ?" dans les commentaires → **signal fort de demande**
- Exemple donné : couleur de l'année = **marron** → se focaliser sur articles marrons

### Technique 3 — Vinted directement
- Moins recommandé sans compte établi (feed pas personnalisé)
- Utile une fois que tu as un vrai fil d'actu

## Logique de sous-niches

Une niche peut se décomposer :
- Niche corporate → sous-niche pantalons / hauts / ceintures / etc.
- Avantage : focus = meilleur référencement + feed Vinted cohérent

## Process de validation d'un produit (AliExpress)

1. Repérer le haut/vêtement sur TikTok ou Pinterest
2. Screenshot de la photo → recherche image sur **AliExpress**
3. Trouver le même produit → ex: **11€** sur Ali
4. Vérifier prix marché : Zara = 30€, H&M = 44€
5. Vendre sur Vinted : **lot haut + bas = 60€ facilement**
6. Marge brute : ~49€ sur 11€ investi

## Stratégie niche avancée (source : chaîne YouTube M22)

### Calcul de la valeur marché
- **Valeur marché = (nb acheteurs × budget moyen) ÷ nb vendeurs**
- Exemple : 1000 femmes dépensent 150€/mois en robes → marché = 150 000€
- S'il y a 1500 vendeurs → valeur réelle = 150 000 / 1500 = **100€/vendeur**
- Objectif : trouver un marché où le ratio acheteurs/vendeurs est le plus élevé

### Le concept du "personnage"
- Chaque compte Vinted doit incarner **un personnage** (ex: femme corporate 30 ans)
- Les photos doivent correspondre au personnage → les acheteurs se projettent
- Si tu vends une veste en cuir → le mannequin IA doit être crédible pour la cible
- Si tu vends des robes à fleurs pour femmes 50 ans → mannequin mature, pas une gamine de 15 ans

### Process de test & scale
1. **Test** : poster 30 annonces d'une niche
2. **Validation** : si 5+ ventes → la niche est validée
3. **Scale** : poster 10-15 annonces/jour dans cette niche
4. **Affinage** : couper les produits qui marchent pas, garder que les winners
5. **Sous-niche** : robes à fleurs → seules les roses marchent ? → ne vendre que ça
6. **Duplication** : une fois la niche affinée → dupliquer sur d'autres comptes

### Marge cible
- **Minimum x5** pour être rentable
- Les meilleurs élèves M22 font du **x10 à x30**
- Meilleur produit M22 : polo à 2€ vendu 45€ = **x22** (27 000€ sur un seul produit)

### Astuce étiquette de marque
- Ajouter une étiquette de marque (même inconnue) sur les vêtements → augmente la valeur perçue
- Les gens sur Vinted préfèrent acheter "une marque" qu'un article sans marque
- Service proposé par la warehouse M22 (option payante)

### Ne PAS copier (sauf si tu lowball)
- Copier ce qui existe sans avantage prix = tu te noies dans la masse
- Copier est OK uniquement si tu sources moins cher et que tu vends moins cher que les autres
- Sinon : trouver une niche propre, se différencier
