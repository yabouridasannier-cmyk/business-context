# MULTI-COMPTES — Setup & Technique

> Source : message coach + call 10h30

---

## Technique PC (validée)

- Télécharger **Chrome Dev** : google.com/chrome/dev
- Créer des **nouveaux utilisateurs** dans Chrome Dev (icône profil > Ajouter)
- Chaque profil = cookies séparés = identité indépendante pour Vinted
- Le coach a **14 profils sur un seul PC** sans problème
- Même technique applicable sur téléphone

## Technique numéros de téléphone

- App **OnOff** : numéros virtuels français
- 1 numéro OnOff par compte Vinted
- Pas besoin de racheter des téléphones physiques

## Compte "hold"

- 1 compte validé avec **vrai IBAN + vraies infos**
- Utilisé pour **acheter ses propres produits** (boost crédibilité / avis)
- ⚠️ Clarifier avec coach : ce compte sert aussi à vendre ou uniquement pour achat interne ?

## Résumé setup minimum pour démarrer

1. Chrome Dev installé + 3-5 profils créés
2. OnOff installé + X numéros virtuels
3. 1 compte hold avec vrai IBAN
4. Emails dédiés pour chaque compte (Gmail aliases ou boîtes séparées)
