## STRATÉGIE NICHES (VALIDÉE M22)

### Principe fondamental :
> **1 compte = 1 niche = 1 personnage = 1 avatar**

### Méthode de test (LES 3 JOURS) :
```
1. Poster 5 articles/jour pendant 3 jours sur 1 compte (= 15 articles)
2. Analyser les stats à J+3
3. Relancer TOUS les favoris (voir section Relances)
4. Si ça marche → continuer + étendre l'avatar
5. Si ça marche pas → changer de sous-niche sur le MÊME compte
6. Répéter jusqu'à trouver le produit gagnant
```

### Niches validées par M22 :

| Niche | Marketplace | Niveau |
|-------|-------------|--------|
| Tapis Moyen Âge / tapisseries médiévales | Vinted | ⭐ Exemple concret M22 |
| Tableaux / Strip arts / Pop culture | Vinted | ⭐ Fort potentiel |
| Robes de mariages (mode soirée) | Vinted + Joyetick | ⭐ M22 vend ça |
| Robes médiévales / de soirées | Joyetick (80€+) | ⭐ High-ticket |
| Lustres, lampes, tables décoration | Vinted | Fort potentiel |
| Vestes gothiques / style particulier | Vinted | Niché |
| Vêtements grandes tailles | Vinted | ✅ Marche TRÈS bien |
| Gros tapis / tableaux valeur forte | Joyetick (80€+) | High-ticket |
| Niche mystère (+480€ en 2 jours) | Vinted | ⭐⭐ À découvrir en live |

### Stratégie d'extension d'avatar :
```
Phase 1 : Vestes pour mamie → valider la niche
Phase 2 : Pantalons assortis → même avatar
Phase 3 : Chaussures → même avatar
Phase 4 : Casquettes/bonnets → même avatar

Résultat : achat groupé (4-5 articles par cliente)
```

### Règle "produits similaires" :
- Un produit qui marche → spammer les **articles similaires** sur LE MÊME COMPTE
- Similaires = même style + variations (couleur, coupe, fermeture)
- ⚠️ Un produit qui marche sur un compte ne marchera pas sur un autre (algorithme)

```
1. TROUVER → Pinterest ou Temu

2. VÉRIFIER SUR VINTED
   → Peu de concurrence ?
   → Photos mauvaises ?
   → Des favoris malgré tout ?

3. OPPORTUNITÉ CONFIRMÉE si :
   ✅ Peu de concurrence + mauvaises photos + faves = GO
   ❌ Forte concurrence + belles photos = passer

4. TESTER 3 JOURS (15 articles)

5. ANALYSER → Relancer → Corriger → Décider
```

## SYSTÈME DE RELANCE FAVORIS (OBLIGATOIRE)

### Message validé M22 :
> *"Bonsoir, j'ai vu que vous avez mis mon article en favori. Désirez-vous d'autres informations ? Désirez-vous de négocier le prix ?"*

### Règles :
- Relancer **TOUS** les favoris à J+3
- Taux de réponse : **~10%**
- Ces 10% = données précieuses pour comprendre les objections
- Objections typiques : prix, dimensions, infos manquantes, doute robot, géographie

## PROCESS PHOTOS (ACTUALISÉ)

### Workflow correct :
```
1. Scanner le produit (photo physique ou depuis Temu/AliExpress)
2. Upscaler → upscale.io
3. Screener le résultat
```

|----------------|-----------------|

### Photos portées :
- Utiles en **photo secondaire** (montrer les dimensions)
- Jamais en photo principale pour les accessoires/sacs

## WAKALA / CONFORMITÉ ISLAMIQUE

### Problème :
Le dropshipping classique = vente d'un bien qu'on ne possède pas → interdit islamiquement.

### Solution validée M22 :
1. Contacter les fournisseurs Temu **directement en anglais**
2. Leur demander l'accord pour vendre en dropshipping
3. Leur accord écrit = **mandat (wakala)** = conforme

### Message type à envoyer sur Temu :
> *"Hello, are you ok? If I can do a dropshipping with your products, any duro position?"*

### Résultat attendu :
- La plupart répondent positivement
- L'accord = wakala = transaction islamiquement valide

### Contact dans le groupe M22 :
- Une personne musulmane dans le groupe est dans la même situation
- A trouvé une solution complète avec une "chaîne" dédiée
- **M22 peut faire la mise en relation** → à demander au prochain live
