## SCORING PRODUITS POUR SOURCING

**Reprendre ce scoring pour évaluer nouveaux produits :**

```
Score = (Vues × 0.5 + Favoris × 1.0 + (Prix / 100) × 0.5) / Compte_Saturation

Résultat > 3 = À sourcer immédiatement
Résultat 2-3 = À tester
Résultat < 2 = Passer
```
