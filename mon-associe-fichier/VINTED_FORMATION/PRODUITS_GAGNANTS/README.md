# PRODUITS GAGNANTS — Mode d'emploi

Un produit est "gagnant" quand il a été **vendu au moins 1 fois sur Vinted**.
À ce moment-là, on crée des VARIANTES (même article, couleurs différentes) pour maximiser les ventes.

---

## Structure

```
PRODUITS_GAGNANTS/
└── [NOM_PRODUIT]/
    ├── FICHE_PRODUIT.md     ← produit original (URL fournisseur, prix, marge, description)
    ├── VARIANTES.md         ← variantes couleurs/tailles à lancer
    └── annonces/
        └── [couleur]/
            ├── ANNONCE.docx ← description prête à copier-coller
            └── images/
                ├── face.jpg
                ├── dos.jpg
                └── sol.jpg
```

---

## Règle variantes (M22)

- **1 vente** → sourcer 2-3 variantes couleurs du même article
- **3 ventes** → considérer comme bestseller, créer 5+ variantes
- **0 vente après 3 jours** → archiver, ne pas faire de variantes
