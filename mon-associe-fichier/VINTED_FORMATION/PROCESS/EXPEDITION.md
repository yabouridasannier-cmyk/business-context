## MODÈLE 1 : WAREHOUSE M22 (RECOMMANDÉ POUR DÉBUTER)

### Vue d'ensemble du service

La warehouse M22 est un service de **réception + reconditionnement + réexpédition** pour dropshippers.

| Étape | Responsable | Timing |
|-------|-------------|--------|
| Tu commandes chez Temu/AliExpress | Toi | Jour 0 |
| Livraison à warehouse | Fournisseur | Jour 5-20 |
| Warehouse reçoit + contrôle qualité | M22 | Jour 21 |
| Reconditionnement (pochettes, étiquettes) | M22 | Jour 21-22 |
| Dépôt point relais | M22 | Jour 22-23 |
| Livraison client final | La Poste/Mondial | Jour 23-27 |

### Tarification Warehouse

| Élément | Coût |
|--------|------|
| **Abonnement fixe** | 50€ / 30 jours |
| **Par colis traité** | 2,20€ / colis |

### Calcul Rentabilité Warehouse

**Exemple 1 : 5 ventes/jour (150/mois)**
```
Abonnement : 50€
Colis : 150 × 2,20€ = 330€
Total warehouse : 380€/mois
Coût par vente : 380€ ÷ 150 = 2,53€/vente

Si marge brute = 40€ par article
Marge nette = 40€ - 2,53€ = 37,47€ ✅ Rentable
```

**Exemple 2 : 10 ventes/jour (300/mois)**
```
Abonnement : 50€
Colis : 300 × 2,20€ = 660€
Total warehouse : 710€/mois
Coût par vente : 710€ ÷ 300 = 2,37€/vente

Si marge brute = 40€ par article
Marge nette = 40€ - 2,37€ = 37,63€ ✅ Très rentable
```

**Exemple 3 : 2 ventes/jour (60/mois) — PERTE**
```
Abonnement : 50€
Colis : 60 × 2,20€ = 132€
Total warehouse : 182€/mois
Coût par vente : 182€ ÷ 60 = 3,03€/vente

Si marge brute = 40€ par article
Marge nette = 40€ - 3,03€ = 36,97€ ✅ Rentable MAIS agilité faible

Recommandation : Attendre 5+ ventes/jour avant warehouse
```

```
Jour 23-27 : Client récupère en point relais

- Client reçoit notification
```

### Avantages Warehouse

✅ **Zéro effort opérationnel**
- Pas de stockage personnel
- Pas d'empaquetage manuel
- Pas de gestion courrier

✅ **Professionnalisme**
- Pochettes discrètes Vinted
- Bordereau pré-rempli
- Tracking pro

✅ **Scalabilité rapide**
- Passer de 5 à 50 colis/jour sans surcharge

✅ **Marges conservées**
- 2,20€ par colis négociable à gros volume

### Inconvénients Warehouse

❌ **Coût fixe mensuel**
- 50€/mois même avec 0 vente

❌ **Délai allongé**
- Attendre 20+ jours avant dépôt (vs 2-3 jours manuel)
- Client attend plus longtemps

❌ **Non idéal pour baskets/chaussures**
- Risque casse en transport

## MODÈLE 2 : EXPÉDITION MANUELLE

### Quand l'utiliser

**Idéal pour :**
- Phase démarrage (< 5 ventes/jour)
- Marge très élevée (x10+, peut absorber cout)
- Articles fragiles (tapis, lustres)
- Besoin de contrôle qualité total

### Matériel Minimum Requis

| Élément | Coût unitaire | Quantité | Coût total |
|--------|---------------|----------|-----------|
| Pochettes blanches | 0,20€ | 100 | 20€ |
| Papier bulle | 5€ | 1 rouleau | 5€ |
| Ruban adhésif | 1€ | 2 rouleaux | 2€ |
| Étiquettes adresse | 0,10€ | 100 | 10€ |
| Stylo noir | 0,50€ | 1 | 0,50€ |
| **Total initial** | - | - | **37,50€** |

**Amortissement :** ~20 colis pour couvrir l'investissement initial

### Process Expédition Manuelle

#### Étape 1 : Réception Colis

```
Jour 0 : Tu commandes chez Temu/AliExpress
Jour 5-20 : Colis arrive à TON adresse
```

#### Étape 2 : Vérification et Stockage

```
À réception :
1. Ouvrir colis, vérifier conformité
2. Tester produit (si possible)
3. Ajouter note de remerciement (optionnel)
4. Stocker en attente de vente
```

#### Étape 3 : Empaquetage

```
Pour CHAQUE vente :
1. Prendre produit en stock
2. Emballer dans pochette blanche (ou papier bulle si délicat)
3. Ajouter note "Merci pour votre achat" (augmente favoris)
4. Fermer pochette avec ruban adhésif
5. Peser le colis (importante pour tarif)
```

#### Étape 4 : Étiquette et Adresse

```
1. Imprimer étiquette Vinted (générée auto après vente)
   → La Poste facilite ce process via app Vinted
2. Coller étiquette sur pochette (zone en haut à droite)
3. Vérifier adresse lisible
```

#### Étape 5 : Dépôt Point Relais / Bureau Poste

```
Options possibles :
1. Point relais La Poste (Gratuit)
   - Aller au point relais + déposer
   - Tracking activé immédiat

2. Timbre La Poste (Non-suivi)
   - Imprimer timbre France/International
   - Mettre dans boîte aux lettres
   - Coût : 4,50€-15€ selon poids/zone

Recommandation : TOUJOURS point relais (tracking = confiance client)
```

#### Étape 6 : Notification Client

```
Après dépôt :
1. Aller sur annonce Vinted
2. Cliquer "Marquer comme expédié"
3. Vinted envoie notification auto au client
4. Client reçoit tracking immédiat
```

### Coûts Expédition Manuelle

| Élément | Coût |
|--------|------|
| Pochette | 0,20€ |
| Emballage (papier, ruban) | 0,30€ |
| Étiquette | 0,10€ |
| Timbre/Dépôt La Poste | 4,50€-6,50€ (selon poids) |
| **Total par colis** | **5,10€-7,10€** |

**Comparaison :**
- Warehouse : 2,20€/colis + 50€/mois fixe
- Manuel : 5-7€/colis + zéro fixe

**Seuil rentabilité :**
- Manuel meilleur jusqu'à 10 ventes/mois
- Warehouse meilleur à partir de 25+ ventes/mois

### Avantages Expédition Manuelle

✅ **Coût variable**
- Payes que ce que tu vendes

✅ **Délai court**
- Client reçoit en 2-3 jours (vs 5-7 warehouse)

✅ **Contrôle total**
- Vérifies produit avant envoi
- Peux ajouter petit "merci" (améliore avis)

✅ **Personnalisation**
- Note handmade = plus d'impact

### Inconvénients Expédition Manuelle

❌ **Effort physique**
- Empaqueter chaque colis
- Aller au point relais régulièrement

❌ **Scalabilité limitée**
- Au-delà de 10 colis/jour = trop fastidieux

❌ **Risque stock**
- Dois stocker physiquement chez toi
- Espace limité

## MATRICE DE DÉCISION

### Décision : Warehouse ou Manuel ?

```
Tu as combien de ventes/jour prévues ?

< 2 ventes/jour → MANUEL (coût trop élevé warehouse)
2-5 ventes/jour → MANUEL (économise 50€/mois fixe)
5-10 ventes/jour → MANUEL si tu aimes, WAREHOUSE si scaling
10+ ventes/jour → WAREHOUSE obligatoire (trop de travail manuel)
20+ ventes/jour → WAREHOUSE 100% (indispensable)
```

### Recommandation pour TON CAS

**Situation actuelle :**
- 4 comptes, ~3-5 articles/semaine
- Estimation ventes : 3-8 colis/semaine (0,5-1 par jour)

**Recommandation :**
🟡 **MODÈLE HYBRIDE (court terme)**
```
Semaines 1-4 : MANUEL (trop peu de volume)
Semaines 5-8 : ÉVALUER (si 5+ ventes/semaine → considérer warehouse)
Semaines 9+ : WAREHOUSE (si scalable confirmée)
```

## TIPS POUR AMÉLIORER SATISFACTION CLIENT

✅ **Petit merci handmade**
- Écrire à la main "Merci pour votre achat !" sur papier
- Ajouter dans le colis
- Augmente favoris de 15-20%

✅ **Emballage soigné**
- Pochette blanc propre (pas pliée)
- Pas de scotch visible en excès
- Étiquette bien alignée

✅ **Timing**
- Expédier dans 24h après achat (délai rapide = meilleur avis)
- Marquer immédiatement "Expédié"

✅ **Suivi**
- Laisser tracking dans Vinted
- Client pas d'surprise

❌ **À ÉVITER**
- Réutiliser emballage fournisseur (trop commercial)
- Délai > 3 jours avant expédition (clients impatients)
- Manquer adresse sur étiquette (colis retour)
