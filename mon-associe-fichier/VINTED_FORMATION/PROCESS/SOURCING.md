## ÉTAPE 1 : TROUVER LA SOURCE PRODUIT

### Sources Primaires (Où chercher)

| Source | Cas d'usage | Avantage |
|--------|-----------|----------|
| **Pinterest** | Inspiration générale + tendance | Photos curatisées, tendance visible |
| **TikTok** | Produits hyper-actuels | Viral, réseau de jeunes femmes |
| **Vinted** | Identifier la concurrence | Voir ce qui se vend déjà |
| **Instagram** | Comptes mode / influenceurs | Produits validés par audience |
| **Temu** | Produits low-cost trend | Prix découverte |
| **AliExpress** | Benchmark prix + produit | Même source, comparaison |
| **1688.com** | Producteur direct | Photos portées, prix ultra-bas |

### Process de recherche

**Étape 1a : Pinterest Search**
```
1. Taper "mode femme 2026" ou "blazer beige femme"
2. Chercher images avec vraies personnes (pas mannequins studio)
3. Sauvegarder 5-10 images par thème
4. Noter : pose, couleur, coupe, contexte
```

**Étape 1b : Vérifier sur Vinted**

Important : avant de sourcer, vérifier que le produit se vend déjà sur Vinted.

```
1. Aller sur Vinted.com (français)
2. Chercher "manteau blanc femme" ou produit similaire
3. Regarder : combien de vendeurs ? Note des photos ?
4. Si < 5 vendeurs + photos médiocres = OPPORTUNITÉ
5. Si > 20 vendeurs + belle qualité = trop concurrentiel
```

**Benchmark concurrence :**
- 0-2 vendeurs : niche vierge (risqué mais à tester)
- 3-8 vendeurs : idéal (niche validée, pas saturée)
- 9-15 vendeurs : niche active (peut marcher si différence)
- 15+ vendeurs : très concurrentiel (haut potentiel si meilleur)

1. **AliExpress** (Débutant)
   - Accès facile

2. **1688.com** (Intermédiaire)

   - Minimum de commande (3-5 pièces)

3. **Alibaba** (Volume)
   - Accès fournisseur direct
   - Minima importants (10+ pièces)

4. **Agent M22** (Professionnel)
   - Accessible à 11-20 commandes/jour
   - Tarif : 20kg pour 155$ + frais logistique
   - Pas de limite d'articles
   - Marges surérieures

### Format de lien Obligatoire

**AliExpress :**
```
https://aliexpress.com/item/[ITEM_ID].html
```

❌ JAMAIS :
```
https://aliexpress.com/item/[ITEM_ID].html?sku_id=xxx&shop_id=xxx&pdp_nps=xxx
```

(Les paramètres créent des tracking issues + le lien peut expirer)

**Vérification du lien :**
1. Copier le lien
2. Tester en incognito (pas loggé)
3. Vérifier : page charge, produit visible, prix correct
4. Note : si lien ne fonctionne pas en incognito = risque expiration

## ÉTAPE 3 : ANALYSER LA FAISABILITÉ

### Calcul Marge

```
Exemple : Manteau blanc

Prix d'achat (fournisseur) : 18€
Coûts logistique/warehouse : 2,50€ (estimation)
Coûts Vinted (frais) : 8% du prix de vente
Prix de vente cible : ?

Formule :
Marge nette = (Prix Vente - Prix Achat - Logistique) - Frais Vinted

Vérification :
Prix Vente = 87€
Marge brute = 87 - 18 - 2,50 = 66,50€
Frais Vinted (8% × 87) = 6,96€
Marge nette = 66,50 - 6,96 = 59,54€
Multiplicateur = 87 / 18 = 4.8x ✅

Validation : Marge 59€ > 40€ minimum, Multiplicateur 4.8x > 3x minimum
→ PRODUIT ACCEPTABLE
```

### Checklist Faisabilité

```
☐ Marge ≥ 40€ après frais ?
☐ Multiplicateur ≥ x3 ?
☐ Prix vente cohérent avec compte/niche ?
☐ Fournisseur note ≥ 3.5★ ?
☐ Stock disponible (> 10 unités) ?
☐ Délai livraison ≤ 30 jours ?
☐ Photos fournisseur de qualité ?
☐ Filtre éthique : accepté ? ✅
☐ Concurrence sur Vinted : 3-8 vendeurs optimal ?
```

Si tout coché → **PRODUIT VALIDÉ POUR SOURCING**

## ERREURS COURANTES À ÉVITER

❌ **Erreur 1 :** Sourcer sans vérifier Vinted
→ Produit peut être complètement saturé

❌ **Erreur 2 :** Faire confiance au prix AliExpress sans vérifier niche
→ Produit peut ne pas se vendre (marge insuffisante)

❌ **Erreur 3 :** Garder des liens avec paramètres
→ Lien expire après 30-60 jours

❌ **Erreur 4 :** Sourcer sans comparer 3 fournisseurs
→ Peut passer à côté de 50% meilleur prix

❌ **Erreur 5 :** Ignorer le filtre éthique
→ Risque rejet personnel lors du post
