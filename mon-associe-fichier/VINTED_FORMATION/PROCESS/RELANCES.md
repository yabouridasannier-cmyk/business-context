## IMPORTANCE CRITIQUE DES RELANCES

**Fait M22 :** Relancer les favoris = **50%+ des ventes totales**

| Métrique | Valeur |
|----------|--------|
| Favoris moyen par article | 15-20 |
| Taux réponse relance | 10% |
| Taux conversion réponses | 40% (4 ventes par 10 réponses) |
| **CA par article en relances** | Équivaut aux ventes organiques |

**Exemple concret :**
```
Article avec 100 favoris
Relancer 100 favoris = 10 réponses
10 réponses × 40% conversion = 4 ventes supplémentaires

Si prix article 60€ = 240€ CA des relances uniquement
Sans relances = 0€ (favoris = rien)
```

**Conclusion :** Relancer est obligatoire, pas optionnel.

## MESSAGE VALIDÉ M22

### Texte Exact

```
"Bonsoir, j'ai vu que vous avez mis mon article en favori.
Désirez-vous d'autres informations ? Désirez-vous de négocier le prix ?"
```

### Pourquoi ce message fonctionne

✅ **Courtois** : "Bonsoir" = politesse
✅ **Direct** : Explique qu'on a vu le favori (presque personnalisé)
✅ **Dual CTA** : Deux options = augmente chance réponse
✅ **Ouverture** : "Autres infos" = prétexte pour discussion
✅ **Négociation** : "Négocier le prix" = objection commune identifiée

### Variantes Optionnelles

**Variante 1 : Avec urgence légère**
```
"Bonsoir, j'ai vu que vous aviez mis mon article en favori.
Désirez-vous d'autres photos/infos ? Je propose une petite réduction
pour une décision rapide ? À bientôt ! 😊"
```

**Variante 2 : Si produit rare**
```
"Bonjour, j'ai remarqué votre intérêt pour cet article.
Il est unique et apprécié par plusieurs personnes.
Souhaiteriez-vous finaliser rapidement ? Ouvert à la négociation."
```

**Variante 3 : Si longtemps en favori**
```
"Bonsoir, ce produit est en favori depuis plusieurs jours.
Une question ? Un doute ? Je suis disponible pour discuter.
Réduction possible si achat rapide."
```

## TIMING DES RELANCES

### Planning Optimal

```
Jour 0 : Article publié + boost immédiat

Jour 1-2 : Accumulation favoris naturels
         (ATTENDRE, pas de relance)

Jour 3 : Première relance (Soir 19h-20h)
        Message TOUS les favoris accumulés
        Exemple : Article a 50 favoris
        → Relancer ~50 messages (0,5-1h max)

Jour 4-5 : Lecture des réponses + suivi
          "Vous avez une objection sur le prix ?"

Jour 6-7 : Deuxième relance (UNIQUEMENT ceux qui n'ont pas répondu)
          Message différent : "Dernière chance !"

Jour 8-10 : Relance finale (Nouveaux favoris de la semaine)

Jour 14+ : Archive l'article ET le repost (si pas vendu)
```

### Fréquence Par Compte

```
Compte avec 5 articles actifs
= potentiellement 100-150 favoris/semaine
= 10-15 messages/jour (faisable en 10 min)

Compte avec 16 articles (eva_76800)
= potentiellement 200-300 favoris/semaine
= 30-40 messages/jour (1 niche, c'est trop)
→ Raison de respecialiser ce compte
```

## VOLUME DES RELANCES

### Limite Vinted (Anti-spam)

**Vinted détecte et pénalise :**
- Plus de 100 messages/jour → possible ralentissement visibilité
- Messages "copiés-collés" identiques → détection possible
- Spam word-by-word → compte peut être signalé

### Stratégie Vinte

✅ **Faire :**
- Max 50-60 messages/jour
- Répartir sur 2-3 tranches (9h, 13h, 19h)
- Légères variations du message
- 1 jour d'intervalle entre relance 1 et relance 2

❌ **Éviter :**
- 150+ messages en 1 heure
- Exact même texte 100 fois
- Plusieurs messages à même personne (spammage)

### Calculatrice Volume Sûr

```
Tes articles actifs : X
Favoris moyens par article : Y
Relancer tous les 3 jours

Volume/jour = (X × Y) / 3

Exemple :
8 articles × 20 favoris = 160 favoris total
160 ÷ 3 jours = ~53 messages/jour ✅ SAFE

Limite recommandée : < 60 msg/jour
Si > 60 msg/jour : réduire articles actifs OU attendre ventes
```

## OBJECTIONS COURANTES & RÉPONSES

### Objection 1 : "C'est trop cher"

**Réponse Type :**
```
"C'est vrai que c'est un prix premium, mais la qualité justifie.
Disposé à négocier pour un achat rapide.
Quelle fourchette de prix vous conviendrait ?"
```

**Taux conversion :** ~30% (certains acceptent réduction 5-10%)

### Objection 2 : "Est-ce que c'est du vrai ?"

**Réponse Type :**
```
"Oui, 100% authentique. Neuf avec étiquette.
Satisfait ou remboursé sur Vinted.
Hésitez pas à poser d'autres questions."
```

**Taux conversion :** ~40% (doute levé = vente)

### Objection 3 : "C'est loin de chez moi"

**Réponse Type :**
```
"Pas de souci, j'expédie partout en France.
Point relais à proximité chez vous.
Livraison rapide (2-3 jours)."
```

**Taux conversion :** ~50% (faux problème)

### Objection 4 : "Vous êtes robot ?"

**Réponse Type :**
```
"Pas du tout ! Je suis vendeur réel.
Vous pouvez vérifier mon historique.
Je suis là pour répondre à toutes vos questions 😊"
```

**Taux conversion :** ~60% (authentification = confiance)

### Objection 5 : "Je suis déjà client, pourquoi ce prix ?"

**Réponse Type :**
```
"Bonne question ! Les anciens clients méritent une réduction.
Je propose -10% immédiate si achat aujourd'hui.
Intéressé ?"
```

**Taux conversion :** ~70% (remerciement fidélité)

## GESTION DES RÉPONSES

### Type 1 : "Oui, je suis intéressé"

**Suivi :**
```
1. Proposer 5-10% de réduction IMMÉDIATE
2. Mettre deadline : "valable 24h"
3. Si accepte → finaliser rapidement

Exemple :
"Parfait ! Je vous propose 50€ au lieu de 55€.
Valable jusqu'à demain. Voulez-vous que je baisse l'annonce ?"
```

### Type 2 : "Quel est votre meilleur prix ?"

**Suivi :**
```
1. Demander d'abord leur budget : "Vous pensez à combien ?"
2. Proposer -10-15% du prix
3. Trouver milieu terme

Exemple :
"Honnêtement, quel budget vous aviez en tête ?
Je peux faire un effort de 10% si ça vous aide 😊"
```

### Type 3 : "Merci mais trop cher"

**Suivi :**
```
1. NE PAS insister
2. Noter mentalement (ré-utiliser si repost)
3. Remercier pour temps

Exemple :
"Pas de souci ! Merci d'avoir regardé.
N'hésitez si vous changez d'avis 👍"
```

### Type 4 : "Je vais réfléchir"

**Suivi :**
```
1. Proposer des photos additionnelles
2. Relancer demain si pas de nouvelles
3. À J+2 : nouvelle relance avec réduction

Exemple :
"Bien ! Voulez-vous des photos de dos ou des détails ?
Je suis là pour vous aider à décider 😊"
```

## STRATÉGIE PRIX EN RELANCES

### Règle M22

> **Ne jamais descendre plus de 15% du prix initial.**

### Progression Réduction

```
Prix original : 100€

Jour 3 (relance 1) : "Ouvert à négociation" (no chiffre)
Jour 6 (relance 2) : 95€ (5% off)
Jour 8 (si demande) : 85€ (15% off = max absolu)

Pas plus loin que 15%.
Sinon : absorbe la marge, article non rentable.
```

### Exemple Réel

```
Article : Manteau blanc (prix 87€)
Coût: 18€
Marge brute : 69€

Limite minimum : 87€ - (69€ × 0,15) = 76€ (15% = 11€ off)

Ne JAMAIS descendre en-dessous de 76€
Sinon : perte marge = pas rentable
```

## ERREURS COURANTES À ÉVITER

❌ **Erreur 1 :** Ne pas relancer du tout
→ Perte 50% du CA potentiel

❌ **Erreur 2 :** Relancer dès J+1
→ Vinted détecte spam, article peut être masqué

❌ **Erreur 3 :** Message trop long
→ Personne lit tout, pas de réponse

❌ **Erreur 4 :** Baisser prix sans limite
→ Marge devient négative, article non rentable

❌ **Erreur 5 :** Relancer personnes qui ont dit non
→ Spam, peut trigger report, compte pénalisé
