## ÉTAPE 1 : UPSCALE LA PHOTO (Obligatoire)

### Pourquoi upscaler ?

❌ **Ne JAMAIS publier une photo AliExpress/Temu directement**
- Risque : **strike immédiat** de Vinted (détection marketplace)
- Résultat : article masqué ou compte banni

✅ **Upscale.io = protection légale**
- Transforme la photo techniquement
- Améliore résolution + qualité
- Vinted ne peut pas détecter la source

### Process Upscale.io

```
1. Aller sur upscale.io (gratuit avec limites, ~3€ pour abonnement)
2. Upload photo produit fournisseur
3. Choisir : "Photo enhancer" mode
4. Attendre traitement (~10 secondes)
5. Télécharger photo upscalée
6. IMPORTANT : screener le résultat (voir Étape 2)
```

### Qualité Minimale Requise

| Critère | Minimal | Bon | Excellent |
|---------|---------|------|-----------|
| Résolution | 1080p | 1440p | 2K+ |
| Blur | Aucun | Imperceptible | Net parfait |
| Saturation | Naturelle | Riche | Légèrement riche |
| Noise | < 5% | Imperceptible | Aucun |

**Checklist après upscale :**
- [ ] Résolution visible ≥ 1080p
- [ ] Pas de pixelization
- [ ] Couleurs naturelles (pas trop saturées)
- [ ] Aucun artefact visible

**Si résultat mauvais :**
- Réessayer upscale.io
- Ou passer à étape 3 directement (générer avec IA)

## ÉTAPE 2 : SCREENER LE RÉSULTAT

### Qu'est-ce que "screener" ?

Prendre une **screenshot** de la photo upscalée (pas la télécharger directement).

### Pourquoi ?

- Photo upscalée brute = métadonnées EXIF présentes
- Screenshot = photo "nouvelle" sans trace d'origine
- Vinted détecte métadonnées = masquage possible

### Process Screener (Rapide)

**Sur Mac :**
```
1. Ouvrir photo upscalée dans Navigateur
2. Cmd + Shift + 4 (screenshot sélection)
3. Sélectionner la zone produit
4. Screenshot créée sur Bureau
5. Utiliser cette version pour Vinted
```

**Sur Windows :**
```
1. Ouvrir photo upscalée
2. Snipping Tool (Win + Shift + S)
3. Sélectionner la zone produit
4. Copier/sauvegarder
5. Utiliser cette version pour Vinted
```

**Résultat :** Nouvelle photo sans métadonnées originales ✅

## ÉTAPE 3 : GÉNÉRER IMAGE IA (Persona + Produit)

### Méthode et Générateurs

#### Générateur 1 : Gemini (Gratuit — Recommandé pour débuter)

**Setup initial :**
```
1. Aller sur gemini.google.com
2. Trouver une photo "persona" sur Pinterest
3. Upload la photo
4. Envoyer prompt :
   "Je vais t'envoyer des articles de mode. Tu vas générer
   des photos réalistes de cette femme portant cet article."
5. Valider avec pouce en l'air (Gemini enregistre comme référence)
6. Prompt d'itération :
   "À partir de maintenant, c'est notre photo de référence.
   Ton travail = faire porter tous les habits que je t'envoie,
   en gardant le même fond, la même pose, la même lumière."
```

**Pour chaque produit :**
```
Prompt type :
"Fais porter un [TYPE] [COULEUR] à la femme de la photo de référence.
Les caractéristiques du [TYPE] :
- [Détail 1]
- [Détail 2]
- [Détail 3]
- Longueur : [LONGUEUR]
Garde la pose naturelle, l'ambiance, le fond."
```

**Avantages :**
- Gratuit (compte Google)
- Homogénéité (même persona pour tout un compte)
- Rapide (2 min/photo)

**Inconvénients :**
- Parfois "lisse" (trop parfait)
- Peut générer incohérences parfois

#### Générateur 2 : ChatGPT Plus (20€/mois)

**Process :**
```
1. ChatGPT Plus (abonnement)
2. Upload image produit
3. Prompt : "Réalise un prompt détaillé pour générer une photo
   réaliste destinée à Vinted, avec ce produit porté naturellement."
4. Recopier le prompt → ImageFX ou Midjourney
5. Générer
```

**Avantages :**
- Prompt très détaillé
- Meilleure résolution possibilité

**Inconvénients :**
- Étapes multiples (ChatGPT + générateur image)
- Plus cher (20€/mois)

#### Générateur 3 : M22 API (50€/mois — Meilleur)

**Process :**
```
1. Contact WhatsApp M22
2. Envoi photo produit
3. Reçois photo portée en 30 min
4. Qualité très élevée (serveur IA local du coach)
```

**Avantages :**
- Meilleure qualité au monde
- Génération rapide
- Support direct

**Inconvénients :**
- 50€/mois (coûteux)
- Nécessite volume (rentable à 5+ articles/semaine)

## ÉTAPE 4 : CADRER + OPTIMISER PHOTOS

### Ratios Obligatoires Vinted

| Contexte | Ratio | Dimensions typiques |
|----------|-------|---------------------|
| Annonce (listing) | 4 : 2.5 (rectangle vertical) | 800 × 500 px |
| Profil du compte | 4 : 3 (quasi-carré) | 600 × 450 px |

**Important :** Toujours rectangle, jamais carré.

### Cadrage par Type de Produit

#### Hauts (blazers, pulls, robes courtes)

```
Cadrage idéal : Mâchoire → Mi-cuisse
Personne : Centrée, face ou très léger 3/4
Mains : Une ou deux visibles (authenticité)
Pose : Selfie miroir (naturelle, pas mannequin)
Spacing : 10% espace en haut, 20% en bas
```

#### Pantalons / Jupes

```
Cadrage idéal : Hanche → Pieds (avec espace)
Angle : 3/4 (montre coupe + longueur)
Pieds : Visibles entièrement (chaussures aussi)
Pose : Debout, légère rotation hanche
```

#### Accessoires / Objets (sacs, montres, bijoux)

```
Cadrage idéal : Produit centered, fond neutre
Fond : Blanc OU couleur cohérente (pas arc-en-ciel)
Angle : De face ou légère rotation
Éclairage : Doux, pas trop de contraste
Spacing : Produit occupe 60-80% de l'image
```

#### Tapis / Décoration (grande surface)

```
Cadrage idéal : En situation (sur le sol, dans la chambre)
Contexte : Montrer la taille/couleur en contexte réel
Angle : De haut (légèrement plongée)
Mise en scène : Avec un objet de référence (lampe, meuble)
```

### Checklist Cadrage

```
☐ Ratio respecté (4:2.5 ou 4:3)
☐ Produit bien visible (60-80% de l'image)
☐ Pas de tête entièrement visible (privé + pas naturel)
☐ Mains visibles (si vêtement porté)
☐ Fond homogène / cohérent avec compte
☐ Lumière naturelle / pas trop sombre
☐ Aucun élément chaotique en arrière-plan
☐ Couleurs naturelles (pas sur/sous-saturées)
```

**Règles :**
- Max 5 mots (court, impactant)
- Mots-clés : ce que les gens tapent vraiment
- Pas de majuscules excessives
- Pas de prix/frais (Vinted refuse)

**Templates par type :**

**Vêtement :**
```

Blazer beige premium
Robe fleurie printemps
Trench camel oversize
Bottes cowboy noires
```

**Accessoire :**
```

Ceinture vintage cuir
Écharpe soie beige
```

**Décoration :**
```
Tapis jute naturel 120x180
Lustre rattan vintage

```

**Checklist titre :**
- [ ] ≤ 5 mots
- [ ] Mots-clés inclus
- [ ] Pas de majuscules excessives
- [ ] Pas de prix/frais
- [ ] Describe le produit précisément

### Description Vinted

**Structure : 3-4 lignes maximum**

```
Ligne 1 : [Produit] en [matière] neuf/très bon état

Ligne 2 : Taille [taille] — Couleur : [couleur]

Ligne 3 : [Caractéristique unique — pourquoi l'acheter ?]
Exemples :
- Coupe époustouflante, rend ultra élégante
- Matière super douce, idéale pour l'automne
- Motif fleuri recherché, rare sur Vinted
- Neuf avec étiquette, jamais porté

Ligne 4 : N'hésitez pas à me poser des questions 💬

[Optionnel] Possibilité de nego — tarif déjà avantageux.
```

**Exemple complet :**
```
Manteau blanc femme XS en polyester

Coupe époustouflante oversize, rend ultra élégante et intemporelle.
Poche intérieures + ceinture intégrée. Neuf avec étiquette.

N'hésitez pas à me poser des questions 💬
```

**Checklist description :**
- [ ] État mentionné (neuf/très bon)
- [ ] Taille et couleur clairs
- [ ] 1 phrase qui vend (caractéristique unique)
- [ ] Ton personnel (pas catalogue)
- [ ] CTA "posez questions" (encourage messages)

## ÉTAPE 6 : PUBLICATION + BOOST

### Processus Publication Vinted

```
1. Ouvrir application Vinted (sur téléphone de préférence)
2. Cliquer "Publier une annonce"
3. Upload photo(s) : 1ère = photo principale, puis les autres
4. Catégorie : sélectionner (Vêtements femme / Accessoires / Déco)
5. Taille : sélectionner (si applicable)
6. État : Neuf / Très bon / Bon
7. Titre : copier depuis préparation (max 5 mots)
8. Description : copier (3-4 lignes)
9. Hashtags : copier (5 max)
10. Prix : mentionné SANS frais Vinted (ils se soustraient auto)
11. Cliquer "Publier"
```

### Boost Immédiat (OBLIGATOIRE)

**Règle M22 :**
> Booster UNIQUEMENT si article récolte 20+ vues naturelles en 6h
> Sinon laisser 24h avant boost

**Quand booster :**
- ✅ Article avec 20+ vues organiques
- ✅ Article avec 5+ favoris naturels
- ✅ Best-seller répliqué (manteau blanc, etc.)
- ❌ Articles sans traction (< 10 vues en 24h)

**Comment booster :**
```
1. Ouvrir annonce
2. Menu 3 points → "Promouvoir"
3. Choisir : 7 jours (recommandé)
4. Budget : 2-5€ (test) à 10-20€ (article chaud)
5. Confirmer
```

**Budget boost selon performance :**

| Vues organiques | Budget recommandé | Durée |
|-----------------|-------------------|-------|
| 20-50 vues | 2€ | 3 jours |
| 50-100 vues | 5€ | 7 jours |

## CALIBRATION QUALITÉ D'IMAGE GEMINI — RÈGLE DE L'ENTRE-DEUX

### Le problème
- Images **trop professionnelles** (studio, éclairage parfait, décor scandinave magazine) → se font masquer par Vinted (détection IA, trop lisse, pas authentique)
- Images **trop messy** (lit défait, sol sale, éclairage jaune, angle tordu) → valeur perçue trop basse, le client ne veut pas payer 50-100€

### La règle d'or : L'ENTRE-DEUX
Le sweet spot = **"j'ai pris 30 secondes pour faire cette photo mais mon appart est correct"**

### Directive de style pour les prompts Gemini
Ajouter systématiquement ce bloc dans chaque prompt de génération :

```
STYLE INSTRUCTIONS: This photo should look like a real person's iPhone photo for Vinted, but still appealing. NOT a professional photoshoot (no perfect lighting, no studio setup), but also NOT super messy or ugly. Think: a tidy-ish real apartment, natural daylight from a window, the item is placed with some care but not perfectly styled. Small real-life details are fine (a remote control, a book, a phone charger cable) but the overall image should still make someone want to buy the item. The colors should be slightly warm and natural, not perfectly white-balanced. Slight angle imperfection is good.
```

### Checklist de validation avant publication
- [ ] L'image fait "vraie" (pas de studio, pas de fond blanc parfait)
- [ ] La valeur perçue reste haute (l'article a l'air désirable)
- [ ] Des petits détails de vie réelle sont visibles (télécommande, tasse, câble)
- [ ] L'éclairage est naturel mais pas parfait (légèrement chaud, pas blanc)
- [ ] L'angle n'est pas parfaitement droit (léger décalage)
- [ ] Le décor est rangé mais pas impeccable

### 3 photos par annonce
1. **Photo mise en scène** : l'article dans son contexte d'usage (canapé, chaise, sol) — entre-deux
2. **Photo détail/texture** : close-up pour montrer la qualité — entre-deux
3. **Photo brute** : l'article seul sur surface simple, vue d'ensemble — un peu plus basique

### Red flags Vinted (ce qui déclenche le masquage)
- Trop lisse, pas assez de contraste
- Trop photoshopé / shooting pro
- Fond trop parfait
- Éclairage de studio
- Mise en scène "monde idéal" / trop bien rangé
