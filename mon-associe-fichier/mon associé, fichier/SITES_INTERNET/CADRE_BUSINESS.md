## MODÈLE ÉCONOMIQUE

### Offre
Développement web + acquisition client (site + machine à trafic).
Ce n'est pas "juste un site" — c'est une infrastructure digitale complète.

### Grille tarifaire AGS

| Format | Prix cible | Cas d'usage |
|--------|-----------|-------------|
| One page | 1 500€ minimum | Artisan, indépendant, landing page |
| Site multipage | 2 000€ - 4 000€ | PME, commerce, service |
| Site complet | 4 000€ - 8 000€ | Entreprise, e-commerce, projet ambitieux |

> **Exception Level Survie** : coût de production quasi nul + temps très court → accepter n'importe quel prix (même 250€) pour déclencher la 1ère vente et sortir de la dette. Retour aux prix AGS dès stabilisation.

### Conditions commerciales
- Acompte : **20%** à la signature
- Geste commercial discret : facturer en TTC plutôt qu'en HT (pas de % mentionné)
- Micro-entreprise = pas de TVA → TTC = HT

### Coût de production
- Temps de production : très court (quelques heures)
- Coût direct : quasi nul
- → Marge nette = quasi 100% du prix de vente

## FUNNEL DE VENTE

```
PROSPECTION
    ↓
Prise de contact (physique / avant-salon / relance)
    ↓
R1 — Découverte + Présentation audit
    ↓  [60% no-show sans process de confirmation]
R2 — Proposition commerciale + Closing
    ↓  [50% no-show sans suivi]
SIGNATURE + Acompte 20%
    ↓
PRODUCTION + Livraison
    ↓
RELANCE pour recommandation / upsell
```

### Stats actuelles (17 mars 2026)
- ~200 calls → 15 R1 → 6 présents → 3 R2 → **0 vente**
- Principal problème : 60% de no-show avant R1

## PROCESS ANTI-NO-SHOW (obligatoire à chaque R1 calé)

```
Jour J (quand R1 est pris)
  → Dans l'heure : email/SMS de confirmation (date, heure, lien visio si applicable)

J-1 (la veille)
  → Rappel + teaser : "J'ai commencé à regarder votre présence en ligne,
    j'ai des observations intéressantes à partager demain."

H-2 (2h avant le RDV)
  → Message court : "On se retrouve dans 2h, tout est bon de votre côté ?"
```

## BLOCAGES IDENTIFIÉS EN R1 (et comment les traiter)

| Blocage | Réponse |
|---------|---------|
| "Je veux réfléchir" | Chiffrer l'inaction : "Chaque mois sans site = X clients perdus sur Google." |
| "Je n'ai pas le budget" | Proposer l'offre d'entrée (1 500€ one page) + acompte 20% = 300€ pour démarrer |
| "Mon site actuel me suffit" | Montrer les data : "Votre concurrent sur Google Maps a 4.8★ et 200 avis, vous avez..." |
| "Je ne comprends pas ce que j'achète" | Reformuler en résultat : "Dans 30 jours, quand quelqu'un cherche [votre métier] à [votre ville], c'est vous qui apparaissez." |
