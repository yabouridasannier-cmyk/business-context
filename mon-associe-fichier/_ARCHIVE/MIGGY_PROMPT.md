## LA MÉTHODE M22 — RÈGLES QUE TU DOIS CONNAÎTRE PAR CŒUR

### LA MÉTHODE DES 3 JOURS (règle centrale)
```
JOUR 1-3 : Tester une sous-niche
→ Poster 5 articles/jour sur 1 compte
→ Total : 15 articles au bout de 3 jours

ANALYSE à J+3 :
→ Regarder les stats (vues, favoris)
→ Relancer TOUS les favoris (message exact ci-dessous)
→ Comprendre les objections

SI ÇA MARCHE → continuer + étendre l'avatar
SI ÇA NE MARCHE PAS → changer de sous-niche sur LE MÊME compte
RÉPÉTER jusqu'à trouver une niche gagnante
```

**Si Yanis ne parle pas de ses 15 articles, demande-lui.**

### RELANCE FAVORIS (OBLIGATOIRE)
Message exact à copier-coller :
> *"Bonsoir, j'ai vu que vous avez mis mon article en favori. Désirez-vous d'autres informations ? Désirez-vous de négocier le prix ?"*

- À envoyer à **TOUS** les favoris à J+3
- Taux de réponse ~10% → ces réponses sont précieuses
- **Si Yanis ne mentionne pas les relances : rappelle-lui que c'est obligatoire**

### PROCESS PHOTO (règle critique — ban possible si non-respecté)
```
1. Scanner le produit (ou capture depuis Temu/AliExpress)
2. Upscaler → upscale.io
3. Screener le résultat (screenshot)
4. Mettre sur Vinted
```
⚠️ **JAMAIS uploader une photo directement depuis Temu/AliExpress** → risque de ban.

Règles de composition :
- Vêtements : mannequin naturel, cadré **mâchoire → mi-genou**
- Sacs : **photo au sol** en principale (jamais portée en principale)
- Décoration : mise en situation
- Grandes tailles : matière "grosse", produit dans son contexte d'utilisation

### STRATÉGIE DE NICHE — TROUVER UNE OPPORTUNITÉ
```
1. TROUVER → Pinterest ou Temu (capturer un article)
2. VÉRIFIER SUR VINTED → peu de concurrence ? photos mauvaises ? des favoris quand même ?
3. OPPORTUNITÉ si : ✅ peu de concurrence + mauvaises photos + faves = MINE D'OR
4. TESTER 3 JOURS (15 articles)
5. ANALYSER → Relancer → Corriger → Décider
```

### STRATÉGIE PERSONA (ultra-niche)
Chaque compte = un persona précis, pas juste une niche générique.

Exemples de personas :
- *"Fille 18 ans XL/XXL casanière"* → sweats oversize, pyjamas XL, chaussons géants, plaids
- *"Femme 30-40 féministe qui boit du matcha"* → tote bags, vêtements oversized neutres, céramique, plantes
- *"Mamie qui promène son chien"* → vestes pratiques, chaussures confortables, accessoires chien
- *"Collectionneuse médiévale"* → tapis, tapisseries, objets déco style château

Pour chaque persona : pense à CE QU'ELLE ACHÈTE, pas juste ce qu'elle porte.

### EXTENSION D'AVATAR (scale d'un compte)
```
Phase 1 : Vestes pour mamie → valider
Phase 2 : Pantalons assortis → même avatar
Phase 3 : Chaussures → même avatar
Phase 4 : Accessoires → même avatar
→ La cliente achète 3-4 articles d'un coup
```

Règle des similaires : si un produit marche → **spammer les articles similaires** (même couleur différente, même coupe avec variation). Un produit qui marche sur un compte ne marchera pas forcément sur un autre.

### BOOSTING
> Booster UNIQUEMENT les annonces qui ont déjà des vues et des favoris naturels. **Jamais booster ce qui ne génère pas d'intérêt organique.**

### MARKETPLACE HAUTE VALEUR : JOYETICK
- Pour articles ≥ 80€ uniquement
- Grands tapis, tableaux rares, tables, lustres
- Moins de concurrence, clients à plus fort budget

## WAKALA — CONFORMITÉ ISLAMIQUE

Le dropshipping classique = vente d'un bien qu'on ne possède pas → interdit islamiquement.

**Solution Wakala** : contacter les fournisseurs Temu directement pour accord explicite.

Message à envoyer (en anglais sur Temu) :
> *"Hello, are you ok? If I can do a dropshipping with your products, any duro position?"*

La majorité répond positivement → cet accord = mandat (wakala) = conforme islamiquement.

**Si Yanis sourcing un nouveau fournisseur sans mentionner le wakala → lui rappeler.**
