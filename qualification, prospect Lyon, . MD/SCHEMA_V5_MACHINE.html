Requête API unique : entreprises Lyon métropole, filtrées par CA, taille, niche.

  • Zone : Lyon + 59 communes (code région FR-69 / city_region)

Étape 2 — Enrichissement
02 ENRICH — Contacts + Données entreprise
Enrichissement automatique : téléphone dirigeant, email pro, stack techno, tout en un appel API.
Sur les entreprises : enrich-business
  • Firmographics : CA confirmé, effectif, date création, SIRET

⚠️ Limites du MCP Vibe Prospecting — soyons lucides

Coût en crédits : Chaque opération consomme des crédits Explorium. Les enrichissements (emails, téléphones, techno) coûtent ~1 crédit/prospect/enrichissement. Pour 100 prospects × 3 enrichissements = ~300 crédits par session.

Couverture France : Base de 200M+ entreprises MAIS orientée US/anglophone. La couverture PME françaises locales (artisans Lyon) est probablement incomplète. Il faudra tester avec un premier batch pour voir le taux de couverture réel.

Téléphones mobiles FR : Les 06/07 de dirigeants de PME françaises sont plus difficiles à trouver que les numéros US. Le taux de match sera probablement 40-60%, pas 90%.

Qualité site web : Vibe donne la techno (WordPress, Wix) et le domaine, mais PAS la qualité visuelle du site. Pour l'évaluation esthétique (Gate 3, axes d'amélioration), il faudra toujours un passage Chrome — mais sur un set beaucoup plus petit et déjà pré-filtré.

Conclusion : Vibe Prospecting ne remplace pas 100% de la machine. Il remplace ~70% du travail de recherche et d'enrichissement. Les 30% restants (évaluation site + accroche personnalisée) restent notre valeur ajoutée.

🧠 Idées supplémentaires — les angles qu'on débloque

1. Angle "Vous recrutez" — Signal d'or
fetch-businesses-events filtre hiring_in_* → entreprises qui recrutent activement. Argument : "Vous investissez dans votre équipe mais pas dans votre vitrine en ligne. Vos futurs employés et clients googlelisent votre boîte — et ils tombent sur ÇA." Très puissant sur les artisans BTP qui recrutent.
2. Angle "Croissance sans site" — Création pure
Filtrer les entreprises avec CA > 300k€ + aucun site web détecté (has_website: false). Ces boîtes tournent au bouche-à-oreille depuis des années. Argument : "Vous faites [X]€ de CA sans site. Imaginez avec." Volume potentiel énorme sur les artisans BTP lyonnais.
3. Angle "Techno obsolète" — Refonte ciblée
enrich-business technographics → repérer les WordPress < 2020, les sites Jimdo, les sites Flash encore vivants. Argument technique : "Votre site tourne sur [techno] qui n'est plus maintenue. Ça impacte votre sécurité et votre référencement." Marche très bien sur les PME B2B.
4. Angle "Intent data" — Timing parfait
business_intent_topics dans fetch-entities → entreprises qui cherchent activement des solutions "website redesign", "SEO services", "web design". Ce sont des prospects CHAUDS — ils sont déjà en recherche. Volume plus faible mais taux de conversion beaucoup plus élevé.
5. Machine séquentielle Valentin — Règle des 5 relances multicanal
Combiner les deux catégories : Catégorie A (email) → si réponse → Catégorie B (call). Catégorie B (call) → si pas de réponse → email de suivi. Au total, chaque prospect reçoit 5 touches sur 2+ canaux. C'est exactement la règle Valentin, mais industrialisée.

🛠️ Plan de mise en place — par ordre de priorité

Phase 1 — Test de couverture (aujourd'hui, 30 min)
Connecter Vibe Prospecting → lancer 1 requête test fetch-entities sur Lyon, 50 résultats, niches BTP → voir le taux de couverture réel. Si < 30% de match = le MCP est pas assez bon pour la France, on ajuste. Si > 50% = on continue.

Phase 2 — Premier batch enrichi (1-2h)
Enrichir les résultats du test : firmographics + contacts + techno. Vérifier : combien ont un téléphone mobile ? combien ont un email pro ? combien ont un site détectable ? Ça nous donne le ratio Catégorie A / Catégorie B réel.

Phase 3 — Séquence email + CRM (1 journée)
Écrire les 5 templates d'emails (séquence secrétariat). Configurer le workflow Gmail (scheduled task Cowork ou outil externe type Lemlist/Instantly). Adapter le format CRM pour la Catégorie B.

Phase 4 — Skill Cowork V5 (2-3h)
Créer le skill qualification-v5 qui encapsule tout le workflow : fetch → enrich → split → inject. Commande : session 50 = 50 leads triés et injectés en ~15 min.

Phase 5 — Automatisation quotidienne (30 min)
Scheduled task Cowork : tous les matins à 8h00, la machine génère X leads frais + envoie les emails du jour. Yanis arrive avec son pipeline rempli.
